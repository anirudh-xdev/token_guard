TokenGuard

Start here

Provision a key, then point any OpenAI-compatible SDK at this host.

Auth model: admin secret unlocks this console · tg_ keys identify apps · provider keys still authenticate upstream. Nothing is “open.”
  1. This host is your new base URL.
  2. Create a TokenGuard user key with Provision user (or API).
  3. Keep your provider API key (OpenAI / OpenRouter / Anthropic).
  4. Add headers X-TokenGuard-API-Key, optional provider + session id.
  5. Handle 401 / 400 / 402 / 409 in your client.
